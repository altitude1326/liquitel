Liquitel

Presentation

Liquitel is a telecom operator project that empowers its users through self-governance, transparency, copyleft and opensource.

  • Provide quality telecommunication services at operation costs,
  • Use minimal amount of resources (material, energy)
  • Give all the decision power to the users,
  • Release all kind of production without any copyright,
  • Insure users that their data are never used for commercial purpose,
  • Give priority to open source hardware and software usage,
  • Allow user to participate to the construction of the infrastucture.
